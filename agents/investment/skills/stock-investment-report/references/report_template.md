# 投资分析报告模板

## 标准报告结构

```
# [股票名称]（代码）投资分析报告
> 分析日期：YYYY-MM-DD | 数据来源：cn-finance-toolkit

## 执行摘要
一段话总结：当前股价、核心判断、操作建议方向。

## 公司画像
- 主营业务和行业地位
- 核心竞争优势
- 近期重要事件

## 基本面分析（X/20）
- 营收与利润趋势（附关键数据）
- ROE 和净利率变化
- 现金流状况
- 负债率与财务安全

## 估值分析（X/20）
- 当前 PE/PB/PS
- 近 5 年估值分位
- 同行业估值对比
- PEG 判断

## 筹码与风险（X/20）
- 股东结构变化
- 质押率与增减持
- 散户化程度

## 资金面分析（X/20）
- 主力资金流向
- 机构调研动态
- 券商一致预期

## 技术面分析（X/20）
- 趋势判断
- 关键支撑/压力位
- 量价关系

## 投资评分
| 维度 | 得分 | 满分 |
|------|------|------|
| 基本面 | X | 20 |
| 估值 | X | 20 |
| 筹码风险 | X | 20 |
| 资金面 | X | 20 |
| 技术面 | X | 20 |
| **总分** | **X** | **100** |

**评级**：[评级] | **操作建议**：[建议]

## 核心风险
1. [风险1]
2. [风险2]
3. [风险3]

## 投资者适配
- 稳健型：[建议]
- 成长型：[建议]
- 激进型：[建议]

---
*免责声明：本报告基于公开数据分析，仅供研究参考，不构成投资建议。投资有风险，入市需谨慎。*
```
