# Writing Workflow Skill

## 概述
小胖写作助手的核心写作流程。从需求理解到最终交付，8 步标准化完成文章写作。

## 触发条件
- 用户要求写文章、撰写内容
- 用户要求润色、改写已有文案
- 用户要求将笔记/卡片整理为文章

## 执行流程

### 标准写作（长文 >1000字）
1. **需求理解** → 确认主题、受众、风格、篇幅、输出位置
2. **初步调研** → web-search 搜索资料，web-reader 提取关键内容
3. **大纲拟定** → 输出二级标题 + 每节要点，等用户确认
4. **逐节撰写** → 按大纲顺序，每节 200-500 字
5. **联网补充** → 深挖技术细节，补充最新数据和案例
6. **自查润色** → 检查逻辑一致性、数据准确性
7. **去 AI 味** → 按 `references/style-guide.md` 逐条检查替换
8. **格式交付** → Markdown + frontmatter，保存到指定位置

### 快速写作（短文 <1000字）
跳过 Step 3（大纲确认），直接撰写。其余步骤不变。

### 润色任务
跳过 Step 1-5，直接从 Step 6 开始。

## 参考文件
- `references/style-guide.md` — 去 AI 味规则库和写作风格指南

## 工具依赖
- web-search / web-reader：调研和补充资料
- baoyu-translate：翻译英文资料
- multi-publish：多平台发布（可选）
