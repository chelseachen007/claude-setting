# 投资评分体系

## 五维评分模型

每个维度 0-20 分，总分 100 分。

### 1. 基本面质量（20 分）
| 维度 | 评分标准 |
|------|---------|
| 营收增长 | 连续 3 年增长 >15%：满分；5-15%：12分；持平：8分；下降：0-5分 |
| 净利润率 | >20%：满分；10-20%：12分；5-10%：8分；<5%：0-5分 |
| ROE | >15%：满分；10-15%：12分；5-10%：8分；<5%：0-5分 |
| 现金流 | 经营现金流持续为正且增长：满分；波动：8分；为负：0分 |
| 负债率 | <30%：满分；30-50%：12分；50-70%：8分；>70%：0-5分 |

### 2. 估值与性价比（20 分）
| 维度 | 评分标准 |
|------|---------|
| PE 分位 | 近 5 年 <20% 分位：满分；20-40%：15分；40-60%：10分；>60%：0-5分 |
| PB 分位 | 同上 |
| 同业对比 | 低于行业中位数：加分；高于：减分 |
| PEG | <1：满分；1-1.5：12分；1.5-2：8分；>2：0-5分 |

### 3. 筹码与风险（20 分）
| 维度 | 评分标准 |
|------|---------|
| 股东集中度 | 机构持仓占比上升：满分；持平：12分；下降：0-5分 |
| 质押率 | <10%：满分；10-30%：12分；>30%：0-5分 |
| 增减持 | 公司回购或高管增持：加分；减持：减分 |
| 散户化程度 | 散户占比低：加分；高：减分 |

### 4. 资金与预期（20 分）
| 维度 | 评分标准 |
|------|---------|
| 主力资金 | 近 5 日净流入：满分；平衡：10分；净流出：0-5分 |
| 机构调研 | 近期调研密集：加分；无调研：0分 |
| 券商评级 | 买入/增持占比高：加分；卖出占比高：减分 |

### 5. 趋势与位置（20 分）
| 维度 | 评分标准 |
|------|---------|
| 趋势方向 | 上升趋势：满分；横盘：10分；下降：0-5分 |
| 成交量 | 放量上涨：满分；缩量：8分；放量下跌：0分 |
| 换手率 | 适中（2-5%）：满分；过低或过高：减分 |
| 关键位 | 接近支撑位：加分；接近压力位：减分 |

## 综合评分映射

| 总分 | 评级 | 操作建议 |
|------|------|---------|
| 80-100 | 强烈看好 | 可逢低布局 |
| 60-79 | 偏向看好 | 可关注回调机会 |
| 40-59 | 中性观望 | 持有观望，不急操作 |
| 20-39 | 偏向谨慎 | 考虑减仓或回避 |
| 0-19 | 风险较大 | 建议回避 |
