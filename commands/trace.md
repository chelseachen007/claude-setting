# 追溯主题学习路径

追溯某个主题在知识库中的学习轨迹，生成学习路径图。

## 执行步骤

1. **主题确认**
   - 用户指定要追溯的主题（必需）
   - 如果主题不明确，列出热门主题供选择

2. **时间线扫描**
   - 扫描 `行思录/` 目录中的日记
   - 按时间顺序找出提到该主题的记录
   - 提取关键观点的演变

3. **卡片关联**
   - 在 `卡片/` 目录搜索相关笔记
   - 分析卡片的创建/更新时间
   - 构建主题的知识图谱

4. **输出学习路径**

   ```markdown
   ## [主题] 学习路径

   ### 时间线
   | 日期 | 来源 | 关键内容 |
   |-----|------|---------|
   | 2023-03 | 行思录 | 初次接触，关注X方面 |
   | 2023-06 | 卡片 | 深入学习，记录了Y |
   | 2024-01 | 行思录 | 实践应用，发现Z |

   ### 相关卡片
   - [[卡片1]] - 基础概念
   - [[卡片2]] - 进阶应用
   - [[卡片3]] - 实践案例

   ### 知识演变
   1. **入门阶段**：了解基本概念
   2. **深入阶段**：研究核心原理
   3. **应用阶段**：项目实践

   ### 待探索方向
   - [ ] 尚未深入的方向A
   - [ ] 可以扩展的方向B
   ```

## 使用场景

- 复盘某个领域的学习历程
- 发现学习盲点
- 规划后续学习方向

## 示例

```
/trace AI学习
/trace Vue开发
/trace 效率提升
```
