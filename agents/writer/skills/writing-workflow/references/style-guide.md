# 写作风格指南 — 去 AI 味规则库

## 第一类：替换规则

以下是 AI 生成文本的典型特征，写作时必须逐条检查并替换。

### 引导语类
| AI 典型表达 | 替换方案 |
|------------|---------|
| 值得注意的是 | 直接说事，删掉引导语 |
| 需要指出的是 | 同上 |
| 不可否认 | 删掉，直接表达观点 |
| 毋庸置疑 | 删掉 |
| 随着XX的发展 | 直接说结论，不要铺垫背景 |
| 在当今XX时代 | 同上 |
| 众所周知 | 删掉，如果真的众所周知就不需要说 |

### 总结语类
| AI 典型表达 | 替换方案 |
|------------|---------|
| 综上所述 | 用具体结论替代空泛总结 |
| 总之 | 同上，或直接省略 |
| 总而言之 | 同上 |
| 归根结底 | 直接给结论 |
| 让我们总结一下 | 删掉，直接写结论段 |

### 结构词类
| AI 典型表达 | 替换方案 |
|------------|---------|
| 首先...其次...最后... | 改为自然过渡，用段落分隔 |
| 一方面...另一方面... | 改为直接对比 |
| 不仅如此 | 删掉，直接展开 |
| 与此同时 | 改为「同时」或直接省略 |
| 更重要的是 | 删掉，用段落重要性自然排序 |

### 夸张修饰类
| AI 典型表达 | 替换方案 |
|------------|---------|
| 革命性的 | 除非真有革命性，否则改用「显著的」「实质性的」 |
| 前所未有的 | 改为具体数据或对比 |
| 令人惊叹 | 删掉或改为具体描述 |
| 无与伦比 | 删掉 |
| 令人印象深刻 | 改为具体说明哪里好 |

## 第二类：结构规则

### 段落节奏
- **长短交替**：不要所有段落都是 3-4 行，要有 1 行短段和 5-6 行长段的交替
- **每段一个观点**：不要在一个段落里堆叠多个论点
- **开头直接**：段落首句就是核心观点，不要先铺垫再转折

### 排比控制
- 连续 3 个以上的排比句打散，改为 1 个核心 + 1 个扩展 + 1 个转折
- 不要所有段落都用相同的句式结构

### 数据呈现
- 数据必须标注来源（「据 XX 报告」而不是「据统计」）
- 用具体数字替代模糊描述（「增长 47%」而不是「大幅增长」）
- 不确定的数据标注「待验证」

## 第三类：语气规则

### 避免说教
- 不用「你应该」「我们必须」「让我们」
- 改为客观陈述或个人经验分享

### 避免 AI 式鼓励
- 不用「这确实是一个很好的问题」
- 不用「让我们一起探索」
- 不在结尾加励志总结

### 保持人味
- 可以有轻微的主观判断
- 可以有不完美的地方（不是每个观点都要面面俱到）
- 可以有明确的态度和立场
