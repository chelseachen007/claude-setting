# 股票多空辩论分析框架

**Extracted:** 2026-03-18
**Context:** 对个股进行投资分析时，避免情绪化决策

## 问题

如何客观评估一只股票是否值得投资？传统分析容易受情绪影响，缺乏结构化论证。

## 解决方案

### 辩论流程

```
1. 看涨方提出3个论点（必须有数据支撑）
2. 看跌方提出3个论点（必须有数据支撑）
3. 双方各反驳对方最强论点
4. 记分卡评分（数据质量、论点强度、反驳质量、时效性、可量化性）
5. 必须有一方获胜——不得"双方都有道理"
```

### 论点类型

| 类型 | 看涨示例 | 看跌示例 |
|------|---------|---------|
| 估值 | PE低于同行 | PE处于历史高位 |
| 增长 | 营收CAGR 30%+ | 增速从30%降至10% |
| 护城河 | 技术/品牌壁垒 | 竞争加剧 |
| 财务 | 现金流健康 | 速动比率<1 |
| 外部 | 政策利好 | 监管风险 |

### 记分卡维度

| 维度 | 评估标准 |
|------|---------|
| 数据质量 | 数据来源是否可靠、具体 |
| 论点强度 | 论点是否有说服力 |
| 反驳质量 | 是否针对实际论点反驳 |
| 时效性 | 信息是否最新 |
| 可量化性 | 是否有数字支撑 |

### 结论模板

```
### 结论：🟢/🔴 [胜方]胜

- 核心理由1
- 核心理由2
- 核心理由3

**投资建议**：
- 买入区间：XX元
- 目标价：XX元
- 止损位：XX元
```

## 触发条件

- 用户请求分析某只股票
- 用户问"XX股票值得买吗"
- 用户进行投资决策前

## 数据来源

- 财报数据（营收、净利润、增长率）
- 估值指标（PE、PB、PEG）
- 行业对比数据
- 股价历史数据
