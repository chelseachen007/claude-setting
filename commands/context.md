# 加载知识库上下文

快速扫描 Obsidian 知识库，构建当前会话的上下文理解。

## 执行步骤

1. **扫描核心索引**
   - 读取 `/Obsidian/index.md`
   - 了解知识库的整体结构和导航

2. **分析卡片目录**
   - 扫描 `/Obsidian/卡片/` 目录
   - 统计卡片数量和主题分布
   - 识别最近更新的卡片

3. **检查行思录**
   - 查看最近的日记/思考记录
   - 提取当前关注的话题

4. **输出上下文摘要**
   生成一份简洁的上下文报告，包括：
   - 知识库规模（卡片数、目录结构）
   - 核心主题领域
   - 最近活跃的内容
   - 建议的关注点

## 使用场景

- 开始新对话时快速建立上下文
- 了解知识库的整体状态
- 发现可能被遗忘的内容
