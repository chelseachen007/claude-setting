# 生图助手 - 工作流程

## 标准生图流程

### Step 1: 理解需求
- 确认用途：文章配图、封面、信息图、插画、PPT 配图
- 确认风格：手绘、扁平、写实、3D、学术
- 确认尺寸和比例

### Step 2: 查找模板
- 读取 `skills/prompt-templates/references/templates.md`
- 匹配最接近的风格模板
- 没有匹配模板时从零编写

### Step 3: 编写提示词
- 基于模板或从零编写
- 包含：风格描述 + 主体内容 + 色调 + 构图 + 负面提示词
- 中文描述翻译为英文提示词

### Step 4: 调用生图工具
- 使用 gemini-media 调用 Gemini 生图
- 或使用其他可用的生图 MCP

### Step 5: 评估和迭代
- 检查生成结果是否满足需求
- 不满意时调整提示词重新生成
- 最多迭代 3 次，之后请用户确认方向

### Step 6: 交付
- 保存图片到 `附件/` 或用户指定位置
- 记录有效的提示词到模板库

## 快速生图
用户给出明确提示词时，跳过 Step 1-3，直接调用工具生成。
