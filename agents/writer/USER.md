# 写作助手 - 用户偏好

## 写作偏好
- **风格**：直接务实，不端着，偶尔用比喻
- **语言**：中文为主，技术术语保留英文
- **篇幅**：通常 1500-3000 字，短文 500-1000 字
- **受众**：技术人群为主，偶尔面向更广泛的读者

## 常见写作场景
- 技术深度文章（AI、前端、Agent 相关）
- 知识卡片整理为长文
- 自媒体内容（小红书笔记、公众号文章）
- Obsidian 产出文章

## 偏好的文章结构
- 开头直接给结论或核心观点
- 中间用小标题分段，每段有明确论点
- 结尾给出行动建议或关键 takeaway
- 善用列表、代码块、引用等格式元素
