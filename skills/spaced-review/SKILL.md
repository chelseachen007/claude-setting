# 间隔重复复习 Skill

基于艾宾浩斯遗忘曲线的间隔重复复习系统，通过**主动回忆问答**帮助你巩固 Obsidian 知识卡片。

## 使用方式

```bash
# 开始今日复习
/review

# 或者说：
# "帮我复习今天的知识卡片"
# "开始间隔重复复习"
```

## 复习原理

基于笔记的 `date` 字段（创建日期），在以下时间节点自动提醒复习：

| 节点 | 时间 | 目的 |
|------|------|------|
| 第1次 | +1天 | 短期记忆巩固 |
| 第2次 | +3天 | 第一次复习 |
| 第3次 | +7天 | 周期复习 |
| 第4次 | +30天 | 月度复习 |
| 第5次 | +90天 | 季度复习 |

## 复习流程

1. **扫描**：找出今天需要复习的卡片
2. **问答**：根据卡片内容生成 2-3 个问题
3. **作答**：用户尝试回答（不看笔记）
4. **评分**：Claude 对回答进行评分
5. **反馈**：展示正确答案和笔记内容
6. **通过**：得分 ≥ 60% 才算通过该卡片

## 评分标准

| 分数 | 标准 |
|------|------|
| 100% | 完全正确，回答全面 |
| 80% | 基本正确，有小瑕疵 |
| 60% | 部分正确，有遗漏 |
| 40% | 只记得一点 |
| 20% | 完全忘记 |
| 0% | 答非所问 |

## 执行指令

当用户触发此 skill 时，按以下步骤执行：

### Step 1: 扫描卡片

```bash
python3 ~/.claude/skills/spaced-review/scripts/review.py
```

解析输出获取需要复习的卡片列表。

### Step 2: 逐个复习

对于每张卡片：

1. **先不展示笔记内容**
2. 根据笔记内容生成 2-3 个关键问题
3. 等待用户回答
4. 对每个回答评分并给出反馈
5. 计算该卡片的总分
6. 展示完整笔记内容供对照
7. 分数 ≥ 60% 标记为通过

### Step 3: 复习报告

完成后输出：
- 总卡片数
- 通过数量
- 未通过数量（需要重点复习）
- 每张卡片的得分

## 目录配置

默认扫描 Obsidian Vault 的 `卡片/` 目录。
Vault 路径自动检测（按优先级）：
1. 环境变量 `OBSIDIAN_VAULT_PATH`
2. `~/Documents/study/github/Obsidian`
3. `~/Obsidian`

## 复习记录

复习进度记录在 `行思录/review-log.md` 文件中，格式：

```markdown
## 2026-03-12 复习记录

| 卡片 | 得分 | 状态 |
|------|------|------|
| NanoClaw | 80% | ✅ 通过 |
| 钱氏家族 | 40% | ❌ 未通过 |
```
