# 发现笔记间的隐藏连接

分析知识库中的笔记，发现潜在的关联和隐藏的主题连接。

## 执行步骤

1. **输入确认**
   - 如果用户指定了特定笔记/主题，以此为中心分析
   - 如果没有指定，则进行全局模式分析

2. **双向链接分析**
   - 扫描 `[[wikilinks]]` 的使用情况
   - 识别被高频引用的"枢纽笔记"
   - 发现孤立笔记（没有入链和出链）

3. **主题聚类**
   - 分析卡片文件名和标签
   - 识别主题集群（如：技术、历史、方法论）
   - 发现跨领域的潜在连接

4. **输出连接报告**

   ```markdown
   ## 发现的隐藏连接

   ### 枢纽笔记（被多次引用）
   - [[笔记A]] - 被 5 篇笔记引用
   - [[笔记B]] - 被 3 篇笔记引用

   ### 跨领域连接
   - "AI学习" ↔ "认知科学" 通过 [[思维链]] 连接
   - "效率工具" ↔ "知识管理" 通过 [[卡片笔记法]] 连接

   ### 孤立笔记（建议连接）
   - [[孤立笔记1]] - 可与 [[相关笔记]] 连接

   ### 建议新建的连接
   - 在 [[笔记X]] 中添加对 [[笔记Y]] 的引用
   ```

## 使用场景

- 发现知识盲区
- 寻找跨领域灵感
- 优化知识网络结构

```

/connect 我的效率方法论
```
