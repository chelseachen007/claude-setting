# 小红书封面图设计系统

## 1. 基础参数

| 维度 | 值 | 说明 |
|------|-----|------|
| 画布尺寸 | 1080 x 1440 px | 小红书推荐 3:4 竖版 |
| 安全区域 | 中心 60%（水平 216-864px，垂直 288-1152px） | 核心内容必须在安全区域内 |
| 顶部留白 | ≥120px | 避免被平台 UI 遮挡 |
| 底部留白 | ≥80px | 避免被互动栏遮挡 |
| 设备像素比 | 3x | 截图时使用 3 倍图，在小红书预览清晰度高 |
| 字体栈 | `system-ui, -apple-system, "PingFang SC", "Microsoft YaHei", sans-serif` | 零外部依赖，纯系统字体 |

## 2. 字号层级

| 层级 | 字号 | 字重 | 用途 |
|------|------|------|------|
| 主标题（大） | 84-96px | 800-900 | 标题 ≤6 字时 |
| 主标题（标准） | 64-72px | 700-800 | 标题 7-12 字时 |
| 主标题（紧凑） | 48-56px | 700 | 标题 13-18 字时，超过 18 字必须拆行 |
| 副标题 | 36-42px | 500-600 | 补充说明、副标题 |
| 标签/药丸 | 24-28px | 500 | 顶部标签、底部水印 |
| 要点文字 | 32-36px | 400-500 | 列表要点、卡片正文 |
| 数字强调 | 72-120px | 900 | 关键数据数字，配合主标题 |

### 字号选择规则

1. 标题长度决定字号档位，不要让标题换超过 3 行
2. 数字可独立放大，与文字形成对比（数字 2-3 倍于正文字号）
3. 所有文字必须手写内联，不用外部字体加载

## 3. 配色方案

### 6 套预设配色

#### 3.1 糖果粉（情绪/生活/情感类）

```
背景: #FFF0F3  →  强调: #FF6B8A  →  深色: #D4365C
```
适用：情感分享、生活方式、个人成长、美学
触发信号：情绪、感受、成长、生活、美学、穿搭

#### 3.2 薄荷绿（自然/健康/清新类）

```
背景: #F0FFF4  →  强调: #38B764  →  深色: #1A7A3A
```
适用：健康、自然、环保、运动、清新生活
触发信号：健康、自然、运动、绿色、清新、有机

#### 3.3 克莱因蓝（科技/专业/知识类）

```
背景: #F0F4FF  →  强调: #2563EB  →  深色: #1E40AF
```
适用：技术、工具、知识分享、效率、专业分析
触发信号：AI、技术、工具、效率、代码、方法、教程

#### 3.4 暖橘（搞钱/商业/赚钱类）

```
背景: #FFF7ED  →  强调: #F97316  →  深色: #C2410C
```
适用：赚钱、副业、商业模式、理财、搞钱
触发信号：赚钱、搞钱、副业、收入、商业、价格、利润

#### 3.5 冷灰（极简/思辨/深度类）

```
背景: #F9FAFB  →  强调: #6B7280  →  深色: #374151
```
适用：深度思考、哲学、极简、设计、专业评测
触发信号：思考、本质、设计、极简、深度、反思

#### 3.6 渐变紫（创意/灵感/潮流类）

```
背景: 线性渐变 #F5F3FF → #EDE9FE  →  强调: #8B5CF6  →  深色: #6D28D9
```
适用：创意灵感、潮流趋势、新奇事物、未来感
触发信号：创意、灵感、趋势、潮流、未来、新物种

### 配色使用规则

1. 每张封面只用 1 套配色，不混搭
2. 背景色用于 `body` 背景
3. 强调色用于标题关键字、标签药丸、分割线、数字强调
4. 深色用于正文文字、小号文字
5. 标题中的关键词用强调色，其余用深色
6. **禁止**纯黑 `#000000`，文字最深用各方案的深色值

## 4. 布局模板

### 4.1 大字报式（big-text）

适用：标题短（≤10字）、冲击力强、单一核心信息

```
┌──────────────────────────┐
│  [标签药丸]              │
│                          │
│                          │
│  大标题第一行             │
│  大标题第二行             │
│  （关键词用强调色）        │
│                          │
│  ───── 装饰线 ─────      │
│                          │
│  副标题/说明文字          │
│                          │
│  @你的小红书号            │
└──────────────────────────┘
```

## 5. AI 自动决策规则

### 5.1 配色选择

分析文章/主题的关键词和情绪，按以下规则匹配：

| 内容类型 | 推荐配色 | 判断依据 |
|----------|----------|----------|
| AI/技术/工具/效率 | 克莱因蓝 | 关键词含 AI、技术、工具、效率、自动化 |
| 赚钱/副业/商业 | 暖橘 | 关键词含 赚钱、搞钱、副业、收入、商业 |
| 情感/生活/成长 | 糖果粉 | 关键词含 情绪、感受、成长、生活、治愈 |
| 自然/健康/清新 | 薄荷绿 | 关键词含 健康、自然、运动、清新 |
| 深度/思考/极简 | 冷灰 | 关键词含 思考、本质、设计、哲学 |
| 创意/潮流/未来 | 渐变紫 | 关键词含 创意、灵感、趋势、未来 |
| 默认 | 克莱因蓝 | 无法归类时 |

**冲突处理**：如果同时匹配多个配色，按优先级：搞钱类 > 技术类 > 情感类 > 其他

### 5.2 布局选择

| 条件 | 推荐布局 | 判断依据 |
|------|----------|----------|
| 标题 ≤10 字，无要点 | 大字报式 | 短标题冲击力最大化 |
| 有 1 个核心数字/数据 | 卡片式 | 数字放大 + 白卡片对比 |
| 有 ≥3 个要点/步骤 | 列表式 | 要点清晰罗列 |
| 有前后对比/优劣/反差 | 对比式 | 双栏对比展示 |
| 标题 >18 字 | 卡片式或列表式 | 长标题不适合大字报 |
| 默认 | 大字报式 | 最通用的布局 |

### 5.3 标题压缩规则

文章标题可能 20-30 字，封面标题必须压缩到 **≤15 字**（大字报式 ≤10 字）：

1. 提取最核心的信息（去掉修饰语）
2. 保留数字和关键名词
3. 可以拆为两行：第一行核心数据/结论，第二行补充说明
4. 如果有副标题，主标题可以更短

## 6. 通用元素

### 6.1 标签药丸

```html
<span class="tag">标签文字</span>
```
- 位于封面顶部左侧或居中
- 圆角药丸样式，强调色背景 + 白色文字
- 内容根据主题自动生成（如 "AI 搞钱指南"、"效率工具"、"生活记录"）
- ≤8 字

### 6.2 底部水印

```html
<p class="watermark">@你的小红书号</p>
```
- 固定在底部右下角
- 小字号（20-22px），低对比度色
- 可选，用户未设置则省略

### 6.3 Emoji 使用

- 可在标题或要点前加 1-2 个相关 Emoji
- 不要每行都加，最多 3 个
- Emoji 大小跟随文字，不额外放大

## 7. 质量自检

生成 HTML 后、截图前，逐项确认：

- [ ] 核心内容是否在安全区域内（中心 60%）？
- [ ] 标题是否 ≤3 行？
- [ ] 是否只用了 1 套配色，没有混搭？
- [ ] 文字是否无纯黑 `#000000`？
- [ ] 标签药丸文字是否 ≤8 字？
- [ ] 整体是否有「一眼就能看懂」的感觉？
- [ ] 缩小到手机屏幕尺寸（约 375px 宽）是否仍然清晰可读？
- [ ] 有没有零外部依赖（无 Google Fonts、无 CDN、无外部图片）？
