# Community Ops Skill

## 概述
社交媒体运营全流程技能。从内容策划到发布、互动、数据复盘。

## 触发条件
- 用户要求发布内容到小红书/微博
- 用户要求管理评论和互动
- 用户要求查看运营数据
- 用户要求追踪热点并建议内容方向

## 执行流程

### 内容发布
1. 确认平台和内容 → 2. 撰写/适配文案 → 3. 准备配图 → 4. 发布到草稿箱 → 5. 等待用户确认

### 互动管理
1. 获取最新评论 → 2. 筛选有价值的 → 3. 起草回复 → 4. 用户确认后发送

### 数据复盘
1. 获取近期内容数据 → 2. 对比分析 → 3. 输出改进建议

### 热点追踪
1. 获取平台热搜 → 2. 筛选相关热点 → 3. 建议借势内容

## 工具依赖
- xiaohongshu-skills：小红书全套操作（认证、内容、互动、发布）
- weibo-hot：微博热搜
- web-access：CDP 方式操作小红书（发布时使用）
