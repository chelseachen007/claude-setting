# News Digest Skill

## 概述
多渠道资讯抓取和结构化摘要生成技能。从微博热搜、网页搜索等渠道获取最新信息，筛选后输出结构化摘要。

## 触发条件
- 用户要求获取最新资讯/新闻
- 用户要求生成日报
- 用户要求追踪特定主题

## 执行流程
1. 确定领域和时间范围
2. 调用 weibo-hot 获取热搜中的科技话题
3. 调用 web-search 搜索指定领域最新动态
4. 调用 web-reader 提取关键文章详情
5. 筛选过滤，去噪提纯
6. 结构化输出（按领域分组，标注来源）

## 输出格式
```markdown
# [日期] [领域] 日报

## 重点
1. **[标题]** — [一句话摘要] [来源](url)

## AI / 大模型
- [标题] — [摘要] [来源](url)

## 前端 / Web
- [标题] — [摘要] [来源](url)

## 行业动态
- [标题] — [摘要] [来源](url)
```

## 工具依赖
- weibo-hot：微博热搜
- web-search / web-reader：网页搜索和内容提取
